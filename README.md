# Theme Toggle Button (Light/Dark Mode)

This project implements a simple theme toggle button that switches between light and dark modes using JavaScript. It allows users to toggle the theme by clicking a button, which changes the appearance of the page using a sun/moon icon to indicate the active theme.

## Features
- **Light/Dark Mode Toggle**: Switch between light and dark modes by clicking a button.
- **Icon-Based Toggle**: Sun icon for light mode and moon icon for dark mode.
- **No Text**: Only icons are shown for the theme toggle, no text displayed.
